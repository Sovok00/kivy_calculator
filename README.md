# Kivy калькулятор

Простой калькулятор на Python с использованием Kivy

## Установка

• Клонируйте реп.:
  https://github.com/Sovok00/kivy_calculator.git
  cd kivy_calculator

2. Зависимости:
   pip install -r requirements.txt

3. Запуск:
   • python main.py

Требования:
   • Python 3.6+
   • Kivy 2.0.0+

Особенности:
   • Базовая арифметика(+, -, *, /)
   • Очистка через кнопку "C"
   • Защита от неправильного ввода операторов
